Film Collection App

A work-in-progress film collector app that allows users to search for and display film information, using the Slim PHP framework and a SQL database. The app makes use of The Movie Database (TMDB) API to perform the search, then transforms the data before storing it for display.
Currently, the user can search, add and delete entries as well as view details such as poster, title, release date, director and cast. Other features which could be added include the ability to sort results, predictive search, fuzzy search, adding a 'favourite' list and adding details such as view date and score.

Authentication API

A RESTful API for user authentication built with Node.js, Express.js and MongoDB. It offers secure registration and login features using JSON Web Tokens for authorisation. Key features include user signup with email and password validation, password hashing with bcrypt, user login, JWT-based authentication and demo protected route, comprehensive testing with Jest.

Encryption tech challenge
